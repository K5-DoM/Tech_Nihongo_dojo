# 08. 評価ルーブリック（5軸）

## スコア定義（1〜5）

### 1) 論理性
- 1: 話が飛ぶ、因果関係が不明
- 3: 概ね筋が通るが抜けがある
- 5: 背景→課題→手法→結果→貢献が明確

### 2) 技術的正確さ
- 1: 明確な誤りが多い
- 3: 大枠正しいが厳密性不足
- 5: 用語・因果・制約の説明が正確

### 3) わかりやすさ（非専門向け）
- 1: 専門用語だらけ
- 3: 説明はあるが冗長
- 5: 比喩・噛み砕きが適切で短い

### 4) 敬語・ビジネス日本語
- 1: 砕けすぎ、誤用多数
- 3: たまに不自然
- 5: 一貫して自然

### 5) 明確さ（曖昧・理解困難でないこと）
- 1: 曖昧で理解困難
- 3: 一部具体例あり
- 5: 数値・事例・役割が明確

## 改善提案ルール
- 提案は最大3つ
- 行動ベース（例: 「次回は結果を数値で述べる」）
- 曖昧語を避ける
