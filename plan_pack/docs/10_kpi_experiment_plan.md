# 10. KPI・検証計画

## 10.1 北極星指標（NSM）
**7日継続ユーザー数**

## 10.2 主要KPI
1. 登録率 = 登録数 / LP訪問数
2. 初回完了率 = 面接完了 / 登録数
3. 7日継続率 = 7日以内2回以上利用 / 登録数
4. 有料転換率 = 課金ユーザー / 無料枠消化ユーザー
5. 粗利/ユーザー = 売上 - 決済手数料 - 推定API原価

## 10.3 仮説
- H1: 弱点再提示で継続率が上がる
- H2: 成果物生成で有料転換が上がる
- H3: 評価根拠表示で離脱が下がる

## 10.4 実験設計
- A/B1: 成果物表示位置（終了直後 vs ダッシュボード）
- A/B2: 料金提示タイミング（無料2回目後 vs 3回目後）
- A/B3: フィードバック量（短文 vs 詳細）

## 10.5 Go/No-Go基準（β終了時）
- 登録 50
- 初回完了率 >= 65%
- 7日継続率 >= 25%
- 有料転換率 >= 8%
