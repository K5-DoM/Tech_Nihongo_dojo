# 01. Project Charter（プロジェクト憲章）

## プロジェクト名
**Tech-Nihongo Dojo**  
理系留学生特化 面接トレーニングシステム

## 背景
日本の就職を目指す理系留学生は、技術力が高くても
- 技術説明を日本語で構造化して伝える
- 面接文脈で敬語/ビジネス日本語を維持する
- 非専門面接官向けに難易度を調整する  
ことに苦戦しやすい。

## 目的（Why）
1. 就活成果に直結する反復訓練環境を提供する  
2. ユーザーの改善履歴を可視化し、継続学習を支援する  
3. ポートフォリオとして説明可能な実運用レベルのSaaSを構築する

## 成果目標（3か月）
- MVP公開（テキスト面接 + 5軸評価 + 履歴）
- 初期ユーザー 20〜50名
- 有料転換 5〜15名（チケット課金）
- 面接練習継続率（7日）25%以上

## スコープ（In）
- テキスト模擬面接
- 5軸評価（論理性・技術的正確さ・わかりやすさ・敬語・具体性）
- 弱点再提示（前回指摘の再確認）
- 面接成果物の生成（30秒自己紹介/60秒研究説明/想定深掘りQA）
- チケット決済（Stripe）

## 非スコープ（Out）
- 完全リアルタイム音声面接（MVPでは対象外、β機能として後続）
- 多言語UI（初期は日本語UI）
- 企業向け管理ダッシュボード（B2B機能）

## 主要制約
- 個人開発（1日2〜4時間）
- 固定費を低く維持
- AI評価の誤判定リスク管理が必要

## 成功条件
- ユーザーが「汎用AI」ではなく本サービスを継続利用する理由があること
- KPIで改善ループが回ること
